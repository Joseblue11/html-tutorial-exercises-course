Un desarrollador full stack es el responsable de construir, testear, mantener y manejar una plataforma web en todos los aspectos, esto significa que ambos tanto Front End a nd Back End. Front End es todo el contenido que puede ser visualizado desde la pantalla, eso es, textos, fotos, formulios, videos, colores y todo lo que tiene interacción directa con el usuario, por el otro lado, el Back End es parte de la plataforma que está detrás, eso es, el manejo de la base de datos que esta página almacena, sistema de auntentificación, manejo de contraseñas, ahorro de datos, etc.
